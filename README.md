# Usage
<!-- usage -->
```sh-session
$ npm install -g jerg
$ jerg COMMAND
running command...
$ jerg (-v|--version|version)
jerg/0.0.0 linux-x64 node-v10.15.3
$ jerg --help [COMMAND]
USAGE
  $ jerg COMMAND
...
```
